Firefox 新鮮事

  • 全新功能 選項畫面改為分頁型式。
  • 全新功能 支援 ruby 注釋。
  • 全新功能 下個 ESR（Extended Support Release）將以此版本的 Firefox 為基礎。
  • 變更 使用者名稱和密碼欄位不再支援 autocomplete=off 屬性。
  • 變更 設置 URL fragment 時不做多餘的 % 符號編碼，但取得 fragment 時依 URL 標準傳回 % 符號解碼的結果。
  • 變更 空的正規表達式物件於 RegExp.prototype.source 將回傳「(?:)」而非空字串。
  • 變更 藉由 speculative connection warmup 改善頁面讀取時間。
  • HTML5 在 Web Worker 中也能使用 WebSocket。
  • HTML5 實作 BroadcastChannel API。
  • HTML5 實作適應性圖片用的 srcset 屬性和 <picture> 元素。
  • HTML5 實作 DOM3 KeyboardEvent.code 屬性。
  • HTML5 Mac OS X: 實作一部分 Media Source Extensions (MSE) API，允許 YouTube 進行原生 HTML 5 播放。
  • HTML5 實作 Encrypted Media Extensions (EME) API 以支援加密的 HTML5 影音播放（限 Windows Vista 或更新版本）。
  • HTML5 自動下載 Adobe Primetime 內容解密模組 (CDM) 以透過 EME 播放受 DRM 保護的媒體（限 Windows Vista 或更新版本）。
  • 開發功能 除錯器 UI 可以看見「optimized-out」的變數。
  • 開發功能 XMLHttpRequest 類的 log 在「網頁主控台」中會被特別標明，且能和一般網路請求分別過濾顯示。
  • 開發功能 WebRTC 的 multistream 和 renegotiation 支援。
  • 開發功能 主控台新增 copy 指令。
  • 修正 各種安全性修正。
  • 請參考此版本的完整變更清單。 您可能也想瞭解 前一個版本的更新。
