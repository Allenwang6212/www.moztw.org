Firefox 新鮮事

  • 參考 Firefox 新鮮事。
  • 全新功能 強化追蹤防護：預設標準設定中此功能現在可以封鎖第三方追蹤 cookie 和加密貨幣礦工，而標準設定和選擇嚴格設定都能封鎖數位指紋。
  • 全新功能 封鎖自動播放：除了自動播放聲音的影片以外，現在還能選擇封鎖任何影片自動播放。
  • 全新功能 嶄新的「全新分頁」即將推出，呈現最棒的 Pocket 內容（僅 US 版本有效）。
  • 全新功能 Windows: 將支援 Windows Hello 的 Web Authentication Hmac Secret extension，2019 年 5 月 10 日後釋出的版本即配備有無密碼上網體驗。
  • 全新功能 這一版將支援多種視訊解碼器，網路即時通訊會議服務整合不同客戶的影像時將更趨便利。
  • 全新功能 Windows 10：可設定內容行程的優先級別，意即處理器花越多時間在主動處理的任務上，在背景工作的所花時間就越少（影片和音訊重播除外）。還可以在 Win10 的工作列上輕鬆找到 Firefox 的捷徑。
  • 全新功能 macOS：雙核心處理器（例如 MacBook Pro）將回歸到更有衝力且低耗電，有助提升電池壽命。Finder 現在也能在檔案下載的同時顯示進度了。
  • 全新功能 為改善 JavaScript 最佳化 JIT 編譯器性能，現在 JIT 已經可以支援 ARM64 了。
  • 修正 各種安全性修正。
  • 變更 Firefox 前一版的Plugin Roadmap 中，flash 外掛程式的內容「總是啟用」已移除，從現在起會徵求使用者許可後再播放網站上的 flash 內容。
  • 變更 不再採用 Adobe Flash 播放器以來，Firefox 也不再需要驗證使用者是否在 64-bit 作業系統上跑 32-bit 的 Firefox，減少使用者代理和數位指紋等並提供更進階的隱私同時，下載其他應用程式的體驗也獲得了改善。
  • 變更 Firefox 不再提供預設載入 userChrome.css 和 userContent.css 以增進啟動性能，想使用前述文件來自訂 Firefox 的使用者可以將 toolkit.legacyUserProfileCustomizations.stylesheets preference 設定為 ture，就能重啟此種功能。
  • 企業版 管理 macOS 電腦的企業版系統管理員們，Mozilla 簽署的 package 安裝包即將推出，簡化您的佈署。。
  • 開發功能 我們的行動網頁開發者們，我們已將 WebIDE 遠端除錯功能搬到全新設計的 關於：除錯，藉由 USB 穩當當地在遠端除錯 GeckoView 。
  • 開發功能 網路資訊面板現在可以顯示封鎖的來源，讓開發者充分感受到內容封鎖的威力，還有這版能讓我們為所有使用者送上持續進化的強化追蹤防護。
  • 開發功能 全新的事件監聽器的中斷點功能可讓開發者暫停在不同事件類型的 host，舉凡動畫、DOM、媒體、滑鼠、使用者觸碰、工人，以及其他許多情境類型。
  • 開發功能 Firefox 開發者工具會提供一段音訊，作為現有的文字內容替代方案，表示非文字內容，a11y 面板 check 工具列已進化以協助開發者跟上 Guideline 1.1.。
  • 請參考此版本的完整變更清單。 您可能也想瞭解前一個版本的更新。
