Firefox 新鮮事

  • 全新功能 分享 Hello 通話網址到社交網路。
  • 全新功能 Project Silk: 更順暢的動畫和捲動（Mac OS X）。
  • 全新功能 網頁親和力：支援 ARIA 1.1「switch」角色。
  • 全新功能 啟用下載檔案惡意軟體偵測功能（Mac OS X 和 Linux）。
  • 全新功能

    支援 Unicode 8.0 的膚色 emoji。

  • 變更 停止支援不安全的 SSL v3 網路通訊。
  • 變更 除了一些在白名單中的主機外，全面停用 RC4。
  • 變更 改善下載時的惡意程式偵測，涵蓋 Mac 的一般檔案類型（bug 1138721）。
  • 變更 改善 Mac OS X 顯示虛線的效能（bug 1123019）。
  • HTML5 List-style-type 可接受字串值。
  • HTML5 對來自 dedicated / shared / service Web Worker 的網路請求啟用 Fetch API。
  • HTML5 以符合目前標準的方式處理 CSS transition 和 animationtion 的階層（cascading）。
  • HTML5 實作 <link rel="preconnect"> ，允許在不洩露個人資訊的前提下進行預先連線。
  • HTML5 支援 CSS Scroll Snap Points。
  • 開發功能 在檢測器的 markup 顯示中，以拖曳的方式移動節點。
  • 開發功能 即使關閉工具列，網頁主控台的輸入歷史仍會保留。
  • 開發功能 編輯 cubic-bezier 時，顯示 CSS 動畫用的 timing-function 供挑選。
  • 開發功能 WebSocket 在離線時也能和 localhost 連線。
  • 修正 改善由 IPv6 降級為 IPv4 過程的效能。
  • 修正 修正未完成的下載，因偵測到 HTTP 1.1 傳輸損壞而標為已完成的問題。
  • 修正 修正安全狀態顯示，應忽略先前頁面（非目前頁面）發生的載入。
  • 修正 修正 Hello 通話視窗有時無法開啟的問題。
  • 修正 修正一個可能造成 Flash 無法顯示的問題。
  • 修正 升級至 NSS 3.19.2。
  • 修正 各種安全性修正。
  • 請參考此版本的完整變更清單。 您可能也想瞭解 前一個版本的更新。
