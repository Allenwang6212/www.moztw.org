Firefox 新鮮事

  • 全新功能 用 Hello 分享分頁。
  • 全新功能 其他裝置上 Firefox 帳號同步的分頁會出現在網址列搜尋結果中。
  • 全新功能 工具列新增「已同步的分頁」按鈕。
  • 全新功能 新設定（network.dns.blockDotOnion）在 DNS 層阻擋 .onion 網域。
  • 全新功能 新增 Guarani [gn]（巴拉圭語）語系支援。
  • 修正 正確轉向含有 Unicode 格式 IDN（Internationalized Domain Name）的 URL。
  • 修正 各種安全性修正。
  • 變更 移除分頁群組（Panorama）功能。
  • 開發功能 記憶體工具的快照篩選功能。
  • 開發功能 動畫檢測器可以調整時間軸的播放倍率。
  • 開發功能 網路監控器時間軸會標示出 DOMContentLoaded 和 load 事件的發生時機。
  • 開發功能 網路監控器新增「過濾網址」功能。
  • 開發功能 記憶體工具新增配置堆疊快照的 diff 功能。
  • 開發功能 檢測器的搜尋功能可以找到子頁框的內容。
  • 開發功能 在動畫檢測器的時間軸中，點選動畫便會顯示相關屬性和關鍵影格。
  • HTML5 支援 Push API，推進 Progressive Web Applications 發展。
  • HTML5 支援以 meta 標籤進行 CSP（Content Security Policy）傳遞。
  • HTML5 支援 Web Speech synthesis API。
  • HTML5 ES6 Classes。
