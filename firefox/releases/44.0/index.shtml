Firefox 新鮮事

  • 全新功能 改善憑證錯誤和連線不受信任的警告頁面。
  • 全新功能 若系統有可用的解碼器，啟用 H.264 影片支援。
  • 全新功能 系統不支援 MP4/H.264 時，啟用 WebM/VP9 影片支援。
  • 全新功能 動畫檢測器中，以合成執行緒運行的動畫會加上閃電圖案標示。
  • 全新功能 HTTPS 內容編碼支援 Brotli 壓縮。
  • 全新功能 開發者工具的畫面擷取指令可以選擇像素比例。
  • 修正 觀看影片時，Windows XP 和 Vista 未停用螢幕保護程式（bug 1193610）。
  • 修正 各種安全性修正。
  • 變更 於 Linux 也採用與其他平台相同的字型比對方式，以支援 webfont 的 unicode-range 設定。
  • 變更 滿足新版 Windows 簽章要件，以 SHA-256 憑證簽署 Windows 建置版本。
  • 變更 不再支援 RC4 decipher。
  • 變更 不再將 Equifax Secure Certificate Authority 的 1024 位元根憑證或 UTN - DATACorp SGC 視為信任的安全網站憑證。
  • 變更 更嚴謹的網頁字型驗證。
  • 變更 暫時關閉 Windows 8 和 8.1 的觸控式鍵盤支援。
  • 開發功能 主控台 log 輸出的物件，可使用右鍵選單「儲存為全域變數」暫存。
  • 開發功能 視覺化的動畫工具：
    • 直接在檢測器中檢視／編輯 CSS 動畫關鍵影格。
    • 視覺化調整 cubic-bezier 曲線，操縱動畫隨時間變化的方式。
    • 尋找、暫停頁面上的 CSS 動畫和 transition。
    • 參見 http://devtoolschallenger.com/。
  • 開發功能 視覺化的版面、樣式工具：
    • 尺規工具會顯示在 viewport 邊緣，另新增「測量頁面中的部分」工具，幫助偵測空間與對齊問題。
    • 即時編輯 CSS filter 以預覽及建立陰影、懷舊、……等效果。
    • 參見 http://devtoolschallenger.com/。
  • 開發功能 新增「記憶體」面板，檢測記憶體堆疊使用狀況。
  • 開發功能 Service Workers API。
  • 開發功能 內建 JSON reader，直覺地檢視、搜尋、複製和儲存資料。
  • 開發功能 在除錯器的原始碼檢視，按住 Cmd（Ctrl）並點選，可以跳到函數定義的位置。
  • 開發功能 WebSocket Debugging API。參考相關的附加元件範例。
  • 開發功能 規則面板的樣式會以 authored 文字顯示，且可以連到樣式編輯器進行編輯。
  • 請參考此版本的完整變更清單。 您可能也想瞭解前一個版本的更新。
