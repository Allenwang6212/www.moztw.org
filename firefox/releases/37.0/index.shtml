Firefox 新鮮事

  • 全新功能 以 Heartbeat 使用者評分系統收集 Firefox 使用者回饋。
  • 全新功能 土耳其語系的預設搜尋引擎改為 Yandex。
  • 全新功能 採 HTTPS 進行安全的 Bing 搜尋。
  • 全新功能 以 OneCRL 集中管理已撤銷的憑證，增進對 site impersonation 的防護。
  • 全新功能 對支援 HTTP/2 AltSvc 的伺服器採 opportunistically encrypt 式的 HTTP 資料傳輸。
  • 變更 停用不安全的 TLS 版本以增進安全性。
  • 變更 擴展 SSL 錯誤報告，回報憑證問題以外的錯誤。
  • 變更 TLS False Start 最佳化要求使用 AEAD 建立的 chiper suite。
  • 變更 移除 DSA 支援以改善憑證與 TLS 資料交換的安全性。
  • 變更 改善 Windows WebGL 繪圖效能。
  • HTML5 實作一部分 Media Source Extensions (MSE) API，允許 YouTube 進行原生 HTML 5 播放（限 Vista 以上版本的 Windows）。
  • HTML5 支援 CSS display:contents。
  • HTML5 Worker 執行緒中也能存取 IndexedDB。
  • HTML5 實作 WebRTC SDP/JSEP。
  • 開發功能 對桌面版、Android 版 Chrome 和 iOS Safari 的分頁進行除錯。
  • 開發功能 檢測器新增「動畫」面板，允許對元素動畫進行控制。
  • 開發功能 網路面板的請求檢視新增「安全性」面板。
  • 開發功能 除錯器面板支援 chrome:// 和 about:// URI。
  • 開發功能 網頁主控台增加對 weak cipher 的記錄。
  • 修正 各種安全性修正。
  • 請參考此版本的完整變更清單。 您可能也想瞭解 前一個版本的更新。
