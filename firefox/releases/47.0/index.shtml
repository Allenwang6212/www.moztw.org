Firefox 新鮮事

  • 全新功能 於 Windows、Mac OS X 支援 Google Widevine CDM，自動將 Amazon Video 等影片串流服務的 Silverlight 影片轉為 HTML5 播放。
  • 全新功能 在速度較佳的裝置啟用 VP9 視訊解碼器。
  • 全新功能 未安裝 Flash 時，使用 HTML5 播放內嵌的 YouTube 影片。
  • 全新功能 在「已同步的分頁」側欄檢視、搜尋您智慧型手機、其他電腦上的分頁。
  • 全新功能 可以在 'no-cache' HTTPS 資源使用上一頁、下一頁功能。
  • 全新功能 新增 Latgalu [ltg]（拉脫維亞語）語系支援。根據 Wikipedia，每日有 164,500 人使用這個語言。
  • 修正 各種安全性修正。
  • 變更 移除 FUEL（Firefox User Extension Library）支援，相依的附加元件也將失效。
  • 變更 重設 browser.sessionstore.restore_on_demand 偏好設定為預設值 true 以避免 e10s 效能問題，快就是好！
  • 變更 移除 Firefox「點擊後播放」白名單。
  • 開發功能 「網頁開發者」相關變更。
  • 開發功能 使用「Service Workers」開發者工具檢視、啟動、除錯已註冊的 Service Worker。
  • 開發功能 使用 Service Workers 工具模擬送出 Push 訊息
  • 開發功能 使用 about:debugging 工具中的 service worker「開始」按鈕啟動已註冊的 Service Worker。
  • 開發功能 影響附加元件相容性的變更。
  • 開發功能 支援 ChaCha20/Poly1305 密文組。
  • 開發功能 在適應性設計檢視模式中自訂使用者代理字串。
  • 開發功能 網頁主控台的智慧多行輸入。
  • HTML5 支援 TextTrack 物件 cuechange 事件。
  • HTML5 WebCrypto：PBKDF2 支援 SHA-2 雜湊演算法。
  • HTML5 WebCrypto：支援 RSA-PSS 簽章。
