Firefox 新鮮事

  • 全新功能 隱私瀏覽模式新增「追蹤保護」功能，封鎖會記錄瀏覽行為的特定元素。
  • 全新功能 隱私瀏覽新增「控制中心」畫面，包含安全性和隱私控制項目。
  • 全新功能 在播放聲音的分頁顯示指示圖標，且可以用來切換靜音。
  • 全新功能 WebRTC 相關改進：
    • 支援 IPV6。
    • 新增用來控制 ICE candidate 生成和 IP 揭露的選項。
    • 提供 hook 讓擴充套件允許／拒絕 createOffer/createAnswer。
    • 改善 getUserMedia 中應用程式監控所使用裝置的能力。
  • 全新功能 密碼管理員相關改進：
    • 改善記憶使用者名稱／密碼時機的判斷。
    • 右鍵選單新增編輯、複製使用者名稱和密碼的功能。
    • 隨時皆可匯入來自 Windows 版 Google Chrome 或 IE 的密碼。
  • 變更 改善互動網站進行大量樣式更新的效能。
  • HTML5 實作 ES6 Reflect。
  • HTML5 支援 ImageBitmap 和 createImageBitmap()。
  • HTML5 所有網站皆可使用 HTML5 video 的 Media Source Extension。
  • 開發功能 以分頁檢視 HTML 原始碼。
  • 開發功能 經由 WiFi 進行遠端除錯（不需要 USB 連線或 ADB）。
  • 開發功能 「呼叫堆疊」能辨認來自 setTimeout、DOM 事件和 Promise 處理器等非同步呼叫的順序。
  • 開發功能 WebIDE 的 Firefox OS 模擬器可以設定適用的手機、平板、電視等裝置。
  • 開發功能 檢測器中自訂的 CSS 預設篩選器可以儲存供未來使用。
  • 修正 修正 OS X Yosemite 下載檔案介面變成簡體中文的問題（bug 1089363）。

    • 已知問題

    • 尚未修復 含有 Unicode-format Internationalized Domain Name (IDN) 的 URL 無法正確重新導向，造成找不到伺服器錯誤。
  • 請參考此版本的完整變更清單。 您可能也想瞭解前一個版本的更新。
