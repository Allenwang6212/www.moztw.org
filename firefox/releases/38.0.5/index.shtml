Firefox 新鮮事

  • 參考 Firefox 38.1 新鮮事。
  • 全新功能 使用 Pocket 追蹤文章和影片。
  • 全新功能 在「閱讀模式」以乾淨格式閱讀文章和部落格。
  • 全新功能 在 Hello 對話中分享正在瀏覽的分頁或視窗。
  • 修正 切換分頁時，因 race condition 而停止繪圖的問題（bug 1067470）。
  • 修正 使用 Windows 7 內建 VGA 驅動程式時的顯示效能問題（bug 1165732）。
  • 請參考此版本的完整變更清單。 您可能也想瞭解 前一個版本的更新。
