Thunderbird 38.1.0 包含一些基於 Firefox 38.1.0esr 的程式碼。這也就是為什麼會跳過這麼多版本號的原因。

Thunderbird 新鮮事

  • 修正 從引文複製貼上的文字，換行會消失 (bug 1143570)
  • 修正 新聞群組的群組清單格式有誤，無法跨群組發文 (bug 1151448)
  • 修正 無法透過 Exchange Server (NTLM) 傳送電子郵件 (bug 1174159)
  • 修正 GB2312 編碼中文字元無法正確顯示 (bug 1174580)
  • 修正 Gmail 伺服器指定 imap.gmail.com 或 smtp.gmail.com 時，OAuth2 會驗證失敗 (bug 1176773)

已知問題

  • 尚未修復 停用 Outlook 以及 Eudora 的匯入功能，程式目前無法使用 (bug 1175055)
